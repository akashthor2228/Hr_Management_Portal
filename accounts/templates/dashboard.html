🎓 Campus Recruitment Management System

A web-based platform built using Django to automate campus hiring between students and companies. It provides separate dashboards for Admin, HR, and Students ensuring fast and efficient placement activities.

🎯Key Features

  • User registration and secure role-based login
  • Admin/HR can add, edit, and delete company drives
  • Admin/HR can manage student profiles
  • Students can update qualification, skills, and resume
  • Students can apply only for eligible drives (based on skills / YOP)
  • HR/Admin can track applied students
  • Clean list views with search and filter options

👥User Roles & Permissions

1️⃣ Admin
  • Manage HR and Student users
  • Add / Update / Delete companies
  • View all student applications
  • Complete control over system
2️⃣ HR
  • Create job drives with required skills
  • Update and maintain company details
  • Track applicants for each drive
  • View all student details
3️⃣ Student
  • Update qualification, skills & resume
  • View eligible drives only
  • Apply for preferred company drives
  • Track application status

📦Modules in the System

  • Student Management: Profile, skills, resume, applied drives
  • Company Management: Drive details like skills, YOP, salary, role
  • Application Tracking: Student can apply & HR/Admin can monitor

🛠️Technologies Used

  • Backend: Python, Django
  • Frontend: HTML, CSS, Bootstrap
  • Database: SQLite / PostgreSQL / MySQL
  • Authentication: Django Auth System
This system helps students explore job opportunities and supports companies in finding the best talent efficiently.
